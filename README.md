# Synthax-week3-task1-project
E-commerce Conversion Funnel Analysis

Overview

This project analyzes an e-commerce customer journey from Visit → Product View → Add to Cart → Purchase to identify conversion bottlenecks and provide actionable business insights.

Objectives

- Measure user progression through each funnel stage.
- Calculate conversion rates between stages.
- Identify where the highest customer drop-off occurs.
- Compare performance across device types, traffic sources, and product categories.
- Present findings through an interactive Power BI dashboard.

Tools Used

- SQL (PostgreSQL)
- Power BI
- Microsoft Excel / CSV

Key Results

- Total Visitors: 1,500
- Product Viewers: 1,245
- Add to Cart Users: 672
- Purchasers: 431
- Overall Conversion Rate: 28.7%

Insights

- The largest drop-off occurs between Product View and Add to Cart.
- Mobile users achieved the strongest overall conversion performance.
- Direct traffic delivered the highest conversion rate.
- Electronics was the best-performing product category.

Repository Contents

- "data/" – Source dataset
- "sql/" – SQL queries used for funnel analysis
- "dashboard/" – Power BI dashboard (".pbix")
- "images/" – Dashboard screenshots
- "README.md" – Project documentation

Business Recommendations

- Optimize product pages to reduce drop-off before the cart stage.
- Prioritize high-converting acquisition channels.
- Improve lower-converting product categories through targeted UX and merchandising enhancements.
